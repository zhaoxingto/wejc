# 06 AI任务模板与提示词库

## 6.1 总模板

```text
你必须严格遵守《渠道平台 AI 开发蓝图（完整版）》。

当前任务：
[写清楚任务名]

项目事实：
1. 多租户 tenant_id + shop_id
2. 私域店铺入口 shop_code / scene
3. source_product + channel_product 双层商品模型
4. SKU 多规格
5. 无支付，下单即生成订单
6. 订单异步推送 ERP / 第三方

技术栈：
FastAPI + SQLAlchemy + PostgreSQL + Pydantic v2

输出要求：
1. 先说明你要修改/新增哪些文件
2. 再给出完整代码
3. 再给出测试建议
4. 再给出下一步推荐任务
```

## 6.2 模型任务模板

```text
请实现 [模块名] 的 SQLAlchemy 模型。

要求：
- 符合多租户设计
- 字段命名统一 snake_case
- 包含 created_at / updated_at
- 包含必要索引
- 不要写业务逻辑到 model
- 仅输出该模块相关文件
```

## 6.3 API任务模板

```text
请实现 [接口名]。

要求：
- 只写 router + schema + service 所需代码
- 不得在 router 里直接写 ORM 查询
- 必须通过 service 调用
- 返回统一响应结构
- 补充异常处理
```

## 6.4 下单任务模板

```text
请实现创建订单接口。

必须满足：
- 校验当前店铺上下文
- 校验商品与 SKU 属于当前店铺
- 生成 channel_order
- 生成 channel_order_item 快照
- 创建 order_push_task
- 不能直接同步调用 ERP
```

## 6.5 ERP推送模板

```text
请实现订单推送任务。

要求：
- 从 order_push_task 读取待推送任务
- 组装 ERP 请求体
- 调用外部 API
- 记录 push_log
- 推送失败支持 retry_count
- 不得影响原订单状态
```

## 6.6 跑偏纠正提示词

如果 AI 开始跑偏，直接贴：

```text
你当前输出已经偏离《渠道平台 AI 开发蓝图（完整版）》。
请立刻回到以下约束：
1. 这是私域渠道系统，不是淘宝式平台
2. source_product 不等于 channel_product
3. SKU 必须结构化
4. 下单后创建推送任务，不是同步推送
5. API 不得跳过 Service 层
请基于这些约束重新输出。
```
