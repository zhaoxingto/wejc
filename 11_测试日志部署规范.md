# 11 测试日志部署规范

## 11.1 测试层级

### 单元测试
- Service 层逻辑
- SKU 组合逻辑
- 订单创建逻辑

### 集成测试
- 店铺解析接口
- 商品详情接口
- 下单接口
- 推送任务执行

### 回归测试
- 多租户隔离
- 不同店铺数据隔离
- 推送失败重试

## 11.2 日志规范

必须记录：
- 关键业务异常
- 订单推送请求与返回
- 商品同步操作
- 店铺解析失败
- 人工重推操作

## 11.3 部署规范

建议：
- Docker
- Nginx
- PostgreSQL
- Redis
- Worker 独立进程

## 11.4 环境变量

至少包括：
- DATABASE_URL
- REDIS_URL
- APP_ENV
- SECRET_KEY
- ERP_TIMEOUT
- LOG_LEVEL
