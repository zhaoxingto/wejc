# 10 ERP与第三方接入规范

## 10.1 接入目标

本系统的接入中心必须支持：
- ERP 商家
- 其他系统商家
- 无系统商家

## 10.2 integration 类型

建议：
- erp
- third
- none

## 10.3 商品同步模式

当前优先支持：

**手动同步**

即：
- 商家后台点“同步商品”
- 平台向 ERP 拉取或 ERP 向平台推送
- 写入 source_product
- 再发布为 channel_product

## 10.4 订单推送模式

优先支持：

**自动异步推送**

流程：
```text
客户下单
-> channel_order
-> order_push_task
-> worker
-> 外部系统
-> push_log
```

## 10.5 接入的最小接口能力

外部系统至少应提供：
- 商品拉取接口
- 订单接收接口
- 可选：订单状态回传接口

## 10.6 接入安全

建议至少支持：
- API Key
- 签名
- IP 白名单
- 请求日志

## 10.7 映射规则

如果外部系统有自己的 product_id / sku_code，平台必须保存映射关系，不可丢失。
